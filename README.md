# Traffic Light Controller

## Overview

This project implements a Traffic Light Controller using Verilog HDL based on a Finite State Machine (FSM). The design was simulated, synthesized, and implemented on FPGA.
## Block Diagram
<img width="967" height="457" alt="image" src="https://github.com/user-attachments/assets/a9eeaee3-ff4b-4357-b6a2-a8257a97c23b" />

## Features

- FSM-based Traffic Light Controller
- 7-Segment Countdown Timer
- Pedestrian Button Support
- Clock Divider
- Timer Module
- FPGA Implementation
- Functional Verification

## Tools

- Verilog HDL
- ModelSim
- Vivado

## Project Structure

- RTL Modules
- Testbench
- Simulation Results
- Synthesis Reports
- FPGA Bitstream
